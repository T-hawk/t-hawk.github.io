Cubemunity

September 2019

Github

This was my first and longest project. I started work on it in July of 2018, and continued to work on it until early 2020. The idea was to provide a space for cubers (people who enjoy solivng Rubik's cubes or other similar puzzles) to hangout and chat. It was my first deepdive into rails and I learned a lot about backend as well as front end design.


Landing page

Community page with posts from fellow cubers. Background color of post changed based off of the number of likes the post has.

Timer that would keep track of your fastest time

As you can see the UI heavily features gradients. It definetely works in some places, but looks bad in others. Although it might not look the greatest, this project was a great learning experience, and I learned how to design a website from the ground up. I had an IOS version of the app in the works, but never quite completed it. The github repo for the mobile version is here
