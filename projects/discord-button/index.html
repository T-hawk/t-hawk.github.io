Discord Button

August 2021

Github

A discord bot created to have fun with my friends. The idea is that the button is constantly moving closer to death, and the only way to revive the button is to press it. When the button is pressed, the presser is assigned a rank, and the closer the button is to death, the higher rank the presser is assigned.
