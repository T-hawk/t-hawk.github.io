Picka Movie

July 2020

Github

During the beginning of the pandemic, my family and I spent a lot of time watching movies together. This resulted in some disagreement on what movie to watch. I thought of a simple solution to this problem: democracy. I created an app that allowed you to create a voting session, share the link with your family, and decide a movie within half an hour. Unfortunetly, I never posted all my code to github and the working code was deleted when I got rid of my old computer.
