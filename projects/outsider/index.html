Outsider

August 2020

Github

I worked on this project with my brother. The idea was to recreate the game Insider, a card game about manipulation and wordplay. It is intended to be used on mobile. The app was written in Svelte, a javascript library.


The insider gets a random word and must lead the conversation to that word.
