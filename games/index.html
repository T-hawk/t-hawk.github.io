UV: Feel the Burn

This game was originally intended for IOS, but can be played in browser. This game was created in under 48 hours for a game jam with the theme of radiation. I didn't go with the obvious approach of basing the game around nuclear radiation, but instead based it on UV radiation. In this game, avoid the dangerous beams from the sun whilst dodging crabs and sharks.

Thaumatech

This was a game that I have worked on over the course of 2 years, but never quite finished. The scope was always way too ambitious, and I would get motivation to start working on it, but always quit after a while. The point of the game was to be able to choose between being a wizard and a steampunk. The world was completely based around a conflict between the Thaumancers (wizards) and the Steampunks.

How I create my games

Game Engine

My game engine of choice is Godot for its ease of use and fast speed


Art

I use Aseprite to create all the art for my games. It is simple, reliable, and has a lot of handy features.


Sound Effects

Audacity is my go to for sound effect creation. It's free and has a lot of nice features that make it easy to create sound effects.
